# Practice Application

An experimental Next.js project using GitHub Projects for task planning and development tracking.

## Getting Started

```bash
npm run dev
```

Open http://localhost:3000 to view the application.

## Project Structure

```
src/
├── pages/
│   └── index.js          # Main page
├── components/
│   └── Hello.js          # Sample component
next.config.js            # Next.js configuration
package.json              # Dependencies and scripts
```

## GitHub Projects Integration

This project uses GitHub Projects boards to organize work. Each task is tracked as a card with clear acceptance criteria.
